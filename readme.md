# Chagnassery Tourism Website

## Overview
This website promotes Chagnassery, Kerala, highlighting its rich history, cultural experiences, and tourist attractions.

## Design & Layout Decisions
- **Banner & Navigation:** A consistent header and navigation bar are used for all pages.
- **Responsive Design:** Utilized Bootstrap for a responsive grid system ensuring mobile/tablet compatibility.
- **Color Scheme:** Chosen to reflect the natural and cultural vibrancy of Kerala.

## Development Process
- Created three HTML pages: Home, History, and Visit.
- Embedded an OpenStreetMap iframe on the home page.
- Structured the Visit page using a 7:5 Bootstrap grid layout.
- Made regular Git commits with clear commit messages (see commit log PDF).

## Challenges
- Integrating a responsive static map.
- Balancing visual appeal with performance and validation standards.

## How to Run
Simply open any of the HTML files in a browser to view the website.
